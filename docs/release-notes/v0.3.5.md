## LCDiff v0.3.5

Patch release focused on Linux compatibility for older stable desktops.

### Changes

- Added Ubuntu 22.04 Linux release artifacts so users on glibc 2.35 systems do
  not need a binary built on a newer glibc floor.
- Kept Ubuntu 24.04 and Ubuntu 26.04 Linux artifacts in the release matrix for
  newer distribution baselines.
- Updated the Docker Linux build helpers and release documentation to use
  Ubuntu 22.04 as the oldest supported Linux build floor.
- Kept Check for updates wired to the signed updater path where available and
  the GitHub Release fallback elsewhere.

### Notes

- Arch Linux users can install or upgrade from AUR:

```bash
yay -S lcdiff
```

- Windows 10/11 builds use the Windows release workflow and remain unsigned
  unless signing secrets are configured.
