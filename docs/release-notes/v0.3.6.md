## LCDiff v0.3.6

Patch release focused on a cleaner, denser desktop workspace and clearer product documentation.

### Changes

- Added a compact left workspace rail for switching between View, Compare, and Free text modes.
- Refined pane density and controls so archive trees and Monaco diffs use more of the window.
- Simplified unused UI and Rust layers while keeping the archive, decompile, search, merge, and save contracts intact.
- Reworked the README around the current feature set with screenshots captured from LCDiff.
- Kept Check for updates wired to signed native updater artifacts, with GitHub Releases as the fallback.

### Notes

- Arch Linux users can install or upgrade from AUR:

```bash
yay -S lcdiff
```

- Windows 10/11 builds use the Windows release workflow and remain unsigned unless signing secrets are configured.
