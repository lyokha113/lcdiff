## LCDiff v0.3.8

Patch release restoring editing, filtering, and preference behavior in the desktop workspaces.

### Fixes

- Restored staged editing and saving for supported text entries in View mode while keeping decompiled and signed content read-only.
- Fixed the Compare tree filters so All, Differences, and Identical show their intended entry sets.
- Kept Compare edits when switching tabs by staging Monaco content changes immediately and ignoring model-replacement events.
- Moved installed-font discovery off the UI thread and deferred it until the Editor preferences tab is opened.
- Shows the selected editor font immediately while the installed-font list is still loading.

### Validation

- Native macOS E2E coverage for View editing, Compare filters, and first-open Preferences behavior.
- Browser E2E coverage for immediate View and Compare staging without relying on editor blur.
- Full frontend, Rust, sidecar, branding, and documentation verification gates.
