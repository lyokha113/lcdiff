## LCDiff v0.3.4

Patch release focused on startup clarity and installer reliability.

### Changes

- Redesigned the startup splash so Text, View, and Compare are presented as
  clear first-class modes.
- Removed the redundant "Desktop workspace" label from the splash header.
- Kept Text mode as the direct paste/type comparison path, View as one-source
  inspection, and Compare as the two-source diff and merge workflow.
- Updated macOS and Linux install helpers to stop a running LCDiff process
  before replacing the app.
- Added in-app Check for updates with native signed updater support where
  available and GitHub Release fallback elsewhere.

### Notes

- Arch Linux users can install or upgrade from AUR:

```bash
yay -S lcdiff
```

- Windows 10/11 builds use the Windows release workflow and remain unsigned
  unless signing secrets are configured.
