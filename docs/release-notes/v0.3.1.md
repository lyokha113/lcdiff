## LCDiff v0.3.1

Patch release focused on Preferences clarity and installed editor font handling.

### Changes

- Added concise hover hints across Preferences controls.
- Improved Monaco editor font selection for installed macOS fonts by loading the
  regular font face through the Tauri asset protocol.
- Kept Monaco font metrics fresh when editor font or font size changes.

### Notes

- macOS installed fonts are loaded from the standard user and system font
  folders.
- Arch Linux users can install or upgrade from AUR:

```bash
yay -S lcdiff
```

- Windows 10/11 builds use the Windows release workflow and remain unsigned
  unless signing secrets are configured.
- Builds remain unsigned unless release signing secrets are configured.
