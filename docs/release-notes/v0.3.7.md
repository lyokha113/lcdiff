## LCDiff v0.3.7

Patch release focused on practical in-app guidance for the View, Compare, and Free text workspaces.

### Changes

- Added a first-run quick tour with mode-specific steps for opening sources, inspection, search, merge, save, status, preferences, and shortcuts.
- Tracks tour completion independently for View, Compare, and Free text so each workspace receives guidance on first use.
- Added **Preferences → Misc → Replay quick tour** for revisiting the current workspace tour.
- Moved the Linux Wayland drag-and-drop notice into a dismissible temporary popup instead of the main app layout.
- Fixed tour layering and mode-switch transitions so highlighted workspace regions cannot block tour controls.

### Notes

- Arch Linux users can install or upgrade from AUR:

```bash
yay -S lcdiff
```

- Windows 10/11 builds remain unsigned unless signing secrets are configured.
